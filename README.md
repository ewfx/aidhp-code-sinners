## 🏃 How to Run
1. Clone the repository  
   ```sh
   git clone https://github.com/ewfx/aidhp-code-sinners.git
   ```
   Come inside code using "cd /code"
2. Install dependencies  
   ```sh
   npm i --legacy-peer-deps
   ```
3. Run the project  
   ```sh
   npm run dev
   ```
